# 🔍 Commit Summarizer - Chrome Extension

A lightweight Chrome Extension that uses AI to summarize the latest GitHub (and Bitbucket in future) commits directly on the commit view page.

## 🚀 Features

- 🧠 **AI-Powered Commit Summaries**
  - Displays what changed, the impact, and a brief note
- 📄 **Inline Summaries**
  - Automatically injected above each changed file on GitHub
- 🕒 **Auto Expiry**
  - Summaries stored briefly (48 hrs) to keep it clean
- ✍️ (Coming Soon) Edit/delete summaries, Jira link support, improvement suggestions
